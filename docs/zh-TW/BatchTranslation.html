# 批量翻譯

在英文語言環境下，這個按鈕將被隱藏。

  1. 點擊 一鍵翻譯按鈕，可將下方所有關鍵詞自動翻譯。

  2. 如果關鍵詞不是英文，將會自動翻譯到英文。如果關鍵詞是英文，將會將對應的本地語言翻譯出來並顯示在關鍵詞下方。

  3. 在某些國家語言的環境中，因為本擴展無法判斷你輸入的是英文還是你的本地語言，這裡會有兩個按鈕。一個是一鍵翻譯到英文，另一個是一鍵翻譯到本地語言。
