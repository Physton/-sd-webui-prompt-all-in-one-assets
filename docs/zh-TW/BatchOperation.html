# 批量操作關鍵詞

滑鼠在空白處按住並拖動，框選需要批量操作的關鍵詞，會顯示批量操作面板。

拖動選中的關鍵詞，可以將多個關鍵詞一起拖動到指定的位置。

  1. 複製 按鈕：將選中的關鍵詞複製到剪貼板。

  2. 收藏 按鈕：將選中的關鍵詞收藏到收藏夾中。

  3. 禁用 按鈕：將選中的關鍵詞禁用。

  4. 啟用 按鈕：將選中的關鍵詞啟用。

  5. 刪除 按鈕：將選中的關鍵詞刪除。
