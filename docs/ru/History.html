# История

# Открыть окно истории

  1. Наведите курсор на иконку истории.

  2. Откройте историю.

  3. Каждое изменение, сделанное вами в Prompt, будет сохранено в истории, и вы можете просмотреть и использовать его в любое время.

  4. В истории максимально сохраняется 100 записей. После того, как их количество превысит 100, самая ранняя запись будет автоматически удалена.

# Описание функций

  1. Удалить все записи в истории. При нажатии будут удалены все записи в текущей категории истории (нельзя восстановить).

  2. Удалить выбранную запись в истории. При нажатии будет удалена выбранная вами запись в истории (нельзя восстановить).

  3. Добавить в избранное. При нажатии будет добавлена выбранная вами запись в историю в избранное, где она будет сохранена навсегда.

  4. Скопировать. При нажатии будут скопированы все Prompts из выбранной вами записи в истории в буфер обмена.

  5. Использовать. При нажатии будут очищены все введенные вами Prompts, и вместо них будут использованы Prompts из выбранной вами записи в истории.
