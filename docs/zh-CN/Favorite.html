# 收藏夹

# 打开收藏窗口

  1. 鼠标移动到 收藏图标 上。

  2. 打开 收藏图标。

  3. 你手动收藏的提示词，都会被记录在收藏夹中，除非你手动取消收藏，否则它们将会永久保留。

# 功能说明

  1. 设置名称，输入此条收藏记录的名称，回车保存。

  2. 取消收藏，点击后，会删除你选中的收藏记录（不可恢复）。

  3. 复制，点击后，会将你选中的收藏中的Prompts复制到你的粘贴板。

  4. 使用，点击后，会清空你已经输入的Prompts，并将你选中的收藏中的Prompts填充到输入框中。
